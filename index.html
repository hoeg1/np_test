How to Play

ニャンプレとは

　ニャンプレは、基本的にはただのナンプレです。

　９ｘ９マスの縦と横のラインや３ｘ３の小部屋にはそれぞれ１から９の数字が１度だけ出現します。プレイヤーは画面に表示されているヒントだけを頼りにすべての数字を発見しなければなりません。

　パズルのルールはこれだけですが、このゲームではあなたのプレイに得点をつけます。

　１から９の数のうちより大きな数を先に発見するほど高得点です。プレイ時間も考慮され、素早く解くほどハイスコアが得られます。

　プレイヤーには得点に応じたメダルが与えられ、最高ランクには Nyanplé の Ⓝ が贈られます。

 

プレイの方法

　Startを押すとパズルが自動生成されます。

　最初に面下部の丸いボタンを押して記入したい数字を選びます。そのあとでパズル画面の記入したい場所をクリックすると選んだ数字がその場所に記入されます。

　記入したい数字を選ぶたび推理を補助するために × 記号 が薄く表示されます。これは選んだ数字が入らない場所を表すマークです。

　記入した数字を取り消したいときは × ボタンを選び、パズル画面の消したい数字をクリックします（※ゲームの最初にパズルのヒントとして表示されていた数字は消すことができません）。

 

メダル

　プレイヤーは得点に応じて次のメダルを手に入れることができます。

  • Ⓝ……19万点以上
    獲得したプレイヤーは神 = ﾈｺです！
  • Ⓢ……18万点以上
  • Ⓐ……17万点以上
  • Ⓑ……16万点以上
  • Ⓒ……15万点以上
  • Ⓓ……14万点以上
  • Ⓔ……13万点以上
  • Ⓕ……上記以下の得点

　このゲームの目的は Nyanplé の Ⓝ を獲得することです。基礎点は平均95,000点程度でボーナスは常に基礎点より少ないため Ⓝ メダルの獲得は想像以上に困難です。

　Ⓝ メダルを目指し、できるだけ短時間で、できるだけ９からパズルを解いてください！

 

補助機能

　自動生成された問題は常に仮定法を使わずに解ける問題になっています。プログラムにパズルを作らせると「もしここに１が入ったら、２が入ったら……」という面倒な仮定法を使わないと解けない問題を出力しがちですが、このアプリが提示するパズルに仮定法は不要です。

　丸いボタンのうち「?」は 正解表示ボタン です。これは自動生成に使っている推論装置で１手先読みした結果のうち 確定できる最も小さな数字 が画面にこっそり表示される機能です。（まれに候補が無い場合があります）

　この数字はあなたがそのマスを確定するか他のマスに数字を記入するまで表示されます。言い換えれば、マス目になにかを書き込まない限り提示された候補は画面から消えません。

　そうして提示された数字はそれまでのプレイに矛盾が無いなら 常に正解 です。このアプリが採用している推論装置は仮定法を使わないので見えている数字のみを追いかければ正しい推理だとわかるはずです。

　ただし、もしもあなたのプレイに矛盾が検知されたら、正解表示ボタンはパズルの正解画面と異なるすべてのマスをエラーとして強調表示します。この警告は常に正しいので、消しゴムで消すか違う数字で上書きしてください。

　推論エンジンは１手しか先読みしないので矛盾が見つからないこともあります。プレイに矛盾があるときは上記の × 記号 の表示が奇妙になることがあるので、矛盾の兆候を見逃さないようにしましょう。

　以上の補助機能はゲーム開始から１分以上経つと利用可能になります。使ったら再び１分待たなければ再使用できません。

 

問題の共有

　パズルに正解した場合、あなたがプレイした問題とスコアを他人と共有することができます。

　クリア画面のボタンやハイスコア画面のスコア部分をクリックすると Game ID をコピーできるので、メールやSNSにコピペしてください。他のプレイヤーは「読み込む…」メニューにそのIDをコピペすることであなたのスコアに挑戦できますし、あなたも誰かのスコアに挑戦できます。

　いずれの場合もある問題に挑戦できるのは１回きりです。すべての問題には固有のIDがあり、一度でもプレイした問題のIDは記録されていて、原則としてプレイ済みの問題に再挑戦することはできません。

 

Daily Quest

　この機能と関連するメニューとして「Daily Quest」があります。

　これは全世界で共通の問題を毎日提示する機能で、すべてのプレイヤーが同じ問題に１度だけ挑戦できます。

　ある日の Daily Quest で良い結果を得られたら ID をコピーしてその日が終わる前に友人に自慢しましょう。IDを受け取ったプレイヤーはその日のうちにあなたのスコアを上回らない限り敗北することになります。

 

得点計算の詳細

　冒頭で述べた通りゲームのスコアは「９などの大きな数字を」「なるべく早く見つける」と高得点です。

　得点は次のように計算されます。

 

得点 = 基礎点 + ボーナス

ボーナス = Max(0, 基礎点 - 時間[ms] ÷ 10)

 

基礎点

　基礎点は毎回の手番で発見した数の 三角数 × 手数 です。

　ここで「手数」はクリックした回数で、最初に８１手あり、消しゴム以外の数字を記入するたび１手消費される数値です。

　三角数とは基本的に「記入した数 × (記入した数 + 1) ÷ 2」のことですが、「記入した数」は最初に表示された問題のヒントに応じて変化します。その数を n とし、その数がヒントに使われた回数を h とするなら、基礎点は次に示す p の三角数を手数に掛け合わせた結果の合計です。

 

pn = n + (n × h) ÷ (9 - h)

基礎点n = { pn × (pn + 1) ÷ 2 } × 手数

 

　例えば９と１の h がそれぞれ０で手数が８１なら、同じ一手でも得られる得点 pt は大きく変わります。

 

９ -> { 9 × (9 + 1) ÷ 2 } = 45

45 * 81 = 3,645pt

１ -> { 1 × (1 + 1) ÷ 2 } = 1

1 * 81 = 81pt

 

　なお、上記の得点が有効になるのはその書き込みがあるマス目に対する正解の書き込みで、かつ最後の書き込みのときだけです。あるマスに正しく９を書き込み、消しゴムで消し、また９を書き込む……ということを繰り返しても得点として採用されるのは最後に書き込んだ「９ × その時の手数」だけです。

　極めてレアなケースとして h = 9 であるとき（つまりその数字がすべて問題に使われているとき）は、クリア時に「その三角数の９倍 × クリア時の手数」が加算されます。

 

タイムボーナス

　画面に表示される時間の単位は「秒」が最小ですが内部的にはミリ秒で記録されていて、ミリ秒単位のプレイ時間でボーナスが決まります。パズルをすべて解いたとき、基礎点からその１０分の１を差し引いた得点がボーナスです。別の見方をするなら１秒につき１００点のボーナスが失われます。

 

ボーナス = Max(0, 基礎点 - 時間[ms] ÷ 10)

 

　Ⓝメダルを獲得するには基礎点で１０万５千点以上を出し、３分以内にクリアしなくてはいけません。
